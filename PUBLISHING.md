# Publishing Instructions

## Preparation

- Update version in `package.json`
- `nvm use`
- `npm i`
- `npm run compile` (optional)

## VS Code Marketplace

- `npx vsce package`
- Upload the `.vsix` to [VS Code Marketplace](https://marketplace.visualstudio.com/manage)

## Open VSX Registry

- Upload the same `.vsix` to [Open VSX Registry](https://open-vsx.org/user-settings/extensions)
